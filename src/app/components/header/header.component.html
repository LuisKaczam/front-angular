Como se Cadastrar?

Para um novo profissional se cadastrar, ele deve preencher seu cadastro na aba Cadastrar novo profissional.

Como se Cadastrar?

Para uma nova gestante se cadastrar, ela deve solicitar ao profissi que está acompanhando seu pré-natal que cadastre ela no sistema. Vale lembrar que para que o profissional possa cadastra-lá, o mesmo precisará possuir uma conta no Sisgestante.
